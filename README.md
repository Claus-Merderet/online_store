# Online store API
## Useful links
1. API documentation: http://localhost:8080/api/doc
2. Kafka UI: http://localhost:9000/
## Install:
1. ```bash
   git clone https://github.com/Claus-Merderet/online_store
   ```
2. ```bash
    make build
   ```
3. ```bash
    make up
   ```
4. ```bash
    make bash
   ```
5. ```bash
    composer install
   ```
6. ```bash
    make migrate
   ```
7. ```bash
   make fixtures
   ```
8. ```bash
   make kafka
   ```
